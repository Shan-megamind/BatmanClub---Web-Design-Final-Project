# Neighborhood Batman Club

A community website for everyday heroes inspired by Batman, where members can learn about the club, view existing patrons, and join the community.

## Overview

The Neighborhood Batman Club is a community of individuals who believe in standing up for what's right, helping neighbors, and making a positive impact in their communities - just like Batman, but without the cape and mask.

## Features

- **Dark/Light Theme**: Switch between Batman-inspired dark theme and Bruce Wayne-inspired light theme
- **Responsive Design**: Fully responsive layout that works on all devices from mobile to desktop
- **Accessible Navigation**: Keyboard-friendly navigation with skiplinks and proper focus management

## Pages

### Home
Learn about Batman's core principles and the mission of the Neighborhood Batman Club. This page provides an introduction to the club and its values.

### Patrons
Discover the everyday heroes already making a difference in their communities. Each patron has an expandable card with their story and a button to subscribe to their alerts.

### Join Us
Become a member of the Neighborhood Batman Club by filling out the application form. Share your skills and motivation to be part of the community.

### Profile
Customize your profile settings, including username, personal details, and theme preference.

## Getting Started

1. Navigate the site using the main menu in the header
2. Explore the Patrons page to learn about existing members
3. Fill out the Join Us form to become a member
4. Customize your experience through the Profile page

## Accessibility

This site is built with accessibility in mind, featuring:
- Semantic HTML
- ARIA attributes
- Keyboard navigation
- Reduced motion support
- Color contrast compliance
